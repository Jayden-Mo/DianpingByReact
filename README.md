# DianpingByReact

react仿制大众点评app项目 个人练手项目 喜欢的star一下 不胜感激


## 技术栈

react + react-router + redux + fetch + webpack + ES6


## 项目运行

git clone https://github.com/Jayden-Mo/DianpingByReact.git

cd DianpingByReact

安装依赖  npm install

启动后台提供模拟的后台数据  npm run mock

启动项目  npm run start(运行在本地8080端口)
